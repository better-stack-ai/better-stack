# BTST Docs

This is the documentation app for BTST in the monorepo.

## Development

```bash
pnpm -w install
pnpm -F "@btst/*" build   # ensure the library is built once
pnpm --filter btst-docs dev
```

- The docs import from the local package `@btst/stack` via `workspace:*`.
- `next.config.mjs` sets `basePath: '/docs'` and `transpilePackages: ['stack']`.
- Visit `http://localhost:3000/docs`.

## Build

```bash
pnpm -F "@btst/*" build
pnpm --filter btst-docs build
```

The static output is generated by Next.js in `.next/`. Deploy with Vercel or any Next-compatible host.

## Type tables (fumadocs)

MDX files reference source files for API tables, e.g.:

```mdx
<AutoTypeTable path="../../../packages/stack/src/types.ts" name="BackendPlugin" />
```

Ensure the relative paths stay correct if you move files. Paths are relative to the MDX file location (e.g., from `docs/content/docs/` you need `../../../` to reach the workspace root).
